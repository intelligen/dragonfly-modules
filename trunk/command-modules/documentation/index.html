Dragonfly-modules documentation¶

Welcome to the Dragonfly-modules documentation. Here you will find information about the command-modules stored in the official Dragonfly module repository.

The currently available modules cover areas of usage such as general desktop activities, application-specific control, and demonstrations of how to use the Dragonfly library.

Below you will find a list of the command-modules currently available and their documentation.

Command-modules¶

This section lists the command-modules presently hosted by Dragonfly module repository, sorted by category.

General desktop activities¶

  • Command-module for moving and controlling windows
  • Command-module for taskbar and icon tray access
  • Command-module for direct opening of user-defined targets

Application-specific control¶

  • Command-module for styles in Microsoft Word
  • Command-module for Audacity, an audio-editing application

Demonstration modules¶

  • Command-module example of CompoundRule use
  • Command-module example of MappingRule use
